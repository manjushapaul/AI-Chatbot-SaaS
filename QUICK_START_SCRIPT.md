## üö® **Common Quick Fixes**

### **Bot Not Responding**
```bash
# Check:
1. Bot status is "Active"
2. Knowledge base is connected
3. Test with simple questions first
4. Clear browser cache
```

### **Widget Not Appearing**
```bash
# Check:
1. Code is in correct location
2. No JavaScript errors in console
3. Bot ID is correct
4. Domain is allowed
```

### **Poor Responses**
```bash
# Quick fixes:
1. Add more specific FAQs
2. Use clear, simple language
3. Test with real customer questions
4. Adjust confidence threshold
```
