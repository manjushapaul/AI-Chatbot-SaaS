This page demonstrates how to embed the AI Chatbot widget on any website.

üìã How to Embed

  1. Copy the embed code below
  2. Paste it into your website's HTML, preferably before the closing </body> tag
  3. The chat bubble will automatically appear in the bottom-right corner
  4. Click the bubble to open the chat popup

Embed Code

<script src="https://ai-chatbot-saas-vert.vercel.app/chat.js" data-widget-id="de7ee0f5a456420eae69658896eb4f79"></script>

Note: Replace de7ee0f5a456420eae69658896eb4f79 with your actual widget ID from the dashboard.

üí° Tip: The widget will automatically:
  • Load when the page loads
  • Show a floating chat bubble button
  • Open a chat popup when clicked
  • Work on all devices (desktop, tablet, mobile)

For Different Platforms

Shopify

// Add to theme.liquid before </body> tag
<script src="https://ai-chatbot-saas-vert.vercel.app/chat.js" data-widget-id="YOUR_WIDGET_ID"></script>

React/Next.js

import { useEffect } from 'react';

function ChatWidget() {
  useEffect(() => {
    const script = document.createElement('script');
    script.src = 'https://ai-chatbot-saas-vert.vercel.app/chat.js';
    script.setAttribute('data-widget-id', 'YOUR_WIDGET_ID');
    document.body.appendChild(script);
    return () => document.body.removeChild(script);
  }, []);
  return null;
}

üîç Troubleshooting:
  • If the bubble doesn't appear, check the browser console for errors
  • Make sure your widget ID is correct
  • Ensure the widget is set to "Active" status in the dashboard
  • Check that the bot associated with the widget is also active
